Node.js file uploads

Welcome to a basic demo for Node.js file uploads using multer. The basic idea of this demo is to show the async nature of file uploads using node, express and multer.

Note: Please choose a large file size.




Random numbers generator

Note: You can use this while the file is uploading. It performs an AJAX request to generate 10 random integers between 1 and 100. Check out the Network tab of your Browser dev tools for more info.
